搭建个人博客

发表于 2020-01-22 更新于 2020-02-18 分类于 hexo

详细步骤如下

一、安装node.js,下载地址，Node.js下载 选择windows版本进行下载


下载完成后双击安装，我安装在D盘blog目录下，一直点击下一步即可

  • node.js环境配置，打开电脑属性

  • 检查node.js是否安装成功，有出现如下信息代表安装成功

    二、安装git，下载地址，git下载，选择相应的版本进行下载

  • 下载完成后一路无脑安装，随意在桌面某个位置右键鼠标出现以下菜单安装成功
  • 检查是否成功安装，如出现以下配置表示安装成功

三、github创建项目，如下所示，仓库名称github.io结尾比较合适

四、安装hexo，在自己的电脑里找个盘目录创建blog文件夹，进入blog文件夹，安装命令 npm install hexo -g

hexo -g

  • 检查安装是否成功，有出现以下信息安装成功
  • hexo初始化，输入命令 hexo init,以下两步时间稍微有点长，请耐心等待
    hexo init
  • 安装组件，输入命令 npm install
    npm install
  • hexo初体验，输入命令 hexo g
    hexo g
  • 启动hexo 输入命令 hexo server -p 端口号,端口号若被占用，可以修改
    hexo server -p 5000
  • 打开浏览器输入http://localhost:5000/ 访问本地博客
    http://localhost:5000/

    五、将hexo和github关联起来

  • 配置git账号和邮箱
    git config --global user.name "hkf"
    git config --global user.email "your email"
  • 生成密钥,打开右键鼠标打开 git Bash Here,同上，在输入框种输入命令 ssh-keygen -t rsa -C “你的邮箱”
    ssh-keygen -t rsa -C "你的邮箱"
  • 添加密钥到ssh-agent,输入命令 eval “$(ssh-agent -s)”
    eval "$(ssh-agent -s)"
  • 再输入ssh-add ~/.ssh/id_rsa，添加生成的SSH key到ssh-agent
    ssh-add ~/.ssh/id_rsa
  • 登入github，点击头像下的settings，添加ssh，新建ssh可以，将id_rsa.pub中的内容复制上去

  • 查看ssh是否添加成功
  • 配置deployment，在blog文件夹下打开_config.yml文件，添加以下内容
  • repository值如下

    六、创建博客，在_posts目录下新建markdown文件

  • 博客部署前，先安装拓展程序npm install hexo-deployer-git –save
    npm install hexo-deployer-git --save
  • 提交博客内容,在命令行输入
    hexo d -g
  • 部署成功后，访问地址 http://用户名.github.io，那么将看到生成的文章
